# Optimizing Culling Strategies for Chronic Wasting Disease
Independent term project for MATH 1380 (Mathematical Biology) at the University of Pittsburgh, Spring 2026.

Built a compartmental ODE model to find the optimal culling rate that minimizes Chronic Wasting Disease prevalence in wild deer populations while keeping the population above a viability threshold. 
Derived R₀ using the next-generation matrix method and ran MATLAB simulations across varying culling rates and environmental decay rates to evaluate tradeoffs between competing management objectives.
Methods: Compartmental ODE modeling, basic reproduction number derivation, parameter sensitivity analysis, MATLAB simulation
